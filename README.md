# Budget Wise

Application de gestion de budget personnel développée en Python.

## 📋 Contexte

Projet réalisé dans le cadre du BTS Développement d'Applications Informatiques (DAI) — Togo.

## 🛠️ Technologies

- **Langage** : Python
- **Interface** : CustomTkinter
- **Base de données** : MySQL
- **Architecture** : MVC

## ✨ Fonctionnalités

- Authentification sécurisée (hachage SHA-256)
- Suivi des revenus et dépenses
- Filtrage des transactions par date
- Report automatique du solde d'un mois à l'autre
- Impression des relevés

## 🚀 Installation

\`\`\`bash
git clone https://github.com/angele2007/BudgetWise-BTS.git
cd BudgetWise-BTS
pip install -r requirements.txt
python main.py
\`\`\`
